# FJ-ker 手动冒烟测试

在 PC 服务端已运行、固件已用正确的 Wi-Fi 和服务端 URL 设置烧录后，使用此检查清单。

- [ ] 开机后 2 秒内出现 `FJ-ker` 启动画面。
- [ ] 按下拍摄键（Capture）进入预览，预览以 4 fps 或更高帧率显示可识别场景。
- [ ] 按下确认键（Confirm）开始拍摄/上传，`正在思考` 加载动画会动。
- [ ] 处理完成后出现答案，中文文本和数学公式都清晰可读。
- [ ] 上一页键（PageUp）和下一页键（PageDown）在第一页和最后一页停止；不会循环。
- [ ] 在答案视图按取消键（Cancel）会返回预览。
- [ ] `正在思考` 期间按取消键（Cancel）会返回预览，并向服务器发送 `DELETE /jobs/{id}`。
- [ ] 当 VBAT 低于 3.3 V 时，设备显示低电量画面并拒绝拍摄。
